CMP 170HX research tool - quick setup
=====================================

From the repository root on Linux, run as your normal user:

    ./install.sh
    .venv/bin/cmpunlock profile list

The installer only creates/updates .venv and performs an offline smoke test. It
does not patch firmware, unload a driver, use sudo, or execute the experimental
path.

The complete live-host preflight, apply, recovery, cold-power-cycle, state-clear,
benchmark, and cooling procedure is maintained in README.md. Read it in full
before using any `cmpunlock system` command. This file intentionally does not
duplicate the hazardous command sequence so the two guides cannot diverge.
